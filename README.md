# `@zhur06/prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
yarn add --dev @zhur06/prettier-config
```

OR

```bash
npm install --save-dev @zhur06/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@zhur06/prettier-config"
}
```
